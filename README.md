# Priothy - Premium E-Commerce Website

## Setup Instructions

### 1. Install Python Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run Flask Backend
```bash
python app.py
```

### 3. Open Website
Open `index.html` in your browser or visit `https://priothy.com/wp-admin/`

## Features
- Beautiful animated product showcase
- Interactive order form with Python backend
- Glassmorphism design footer
- Smooth scroll animations
- Responsive design
- Flask API integration
